# Commit Message

When generating a commit message:

- **One sentence only** — no multi-line body
- **English** — all text in English
- **Conventional format** (optional): `type(scope): brief description`
- Keep it concise, descriptive of the change

## Types

`fix`, `feat`, `refactor`, `chore`, `docs`, `style`, `test`

## Example

```
fix(role): allow adding employees when role already has members
```
